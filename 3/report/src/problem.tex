\CWHeader{Лабораторная работа \textnumero 3}

\CWProblem{
Для реализации словаря из предыдущей лабораторной работы необходимо провести исследование скорости выполнения и потребления оперативной памяти. В случае выявления ошибок или явных недочётов требуется их исправить.

Результатом работы должны являться:
\begin{enumerate}
    \item дневник выполнения работы, в котором отражено, что и когда делалось, какие средства использовались и какие результаты были достигнуты на каждом шаге;
    \item выводы о найденных недочётах;
    \item сравнение работы исправленной программы с предыдущей версией;
    \item общие выводы о выполнении лабораторной работы и полученном опыте.
\end{enumerate}

Минимальный набор средств должен содержать утилиту \texttt{gprof} и библиотеку \texttt{dmalloc}, однако допускается использование аналогичных или более развитых инструментов, например \texttt{Valgrind}.
}
