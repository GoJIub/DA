\CWHeader{Лабораторная работа \textnumero 5}

\CWProblem{
Необходимо реализовать поиск поступающих на вход образцов в заранее известном тексте с использованием
суффиксного дерева.

{\bfseries Вариант задачи:} Поиск в известном тексте неизвестных заранее образцов.

Первая строка входа содержит текст. Далее до конца файла следуют строки с образцами. Для каждого
образца, который встретился в тексте, требуется вывести номер образца, двоеточие и позиции всех его
вхождений в порядке возрастания. Нумерация образцов и позиций начинается с единицы.
}
