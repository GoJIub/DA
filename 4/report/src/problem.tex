\CWHeader{Лабораторная работа \textnumero 4}

\CWProblem{
Необходимо реализовать один из стандартных алгоритмов поиска образцов для указанного алфавита.

{\bfseries Вариант алгоритма:} Поиск одного образца, основанный на построении Z-блоков.

{\bfseries Вариант алфавита:} Слова не более 16 знаков латинского алфавита, регистронезависимые.

Искомый образец задаётся на первой строке входного файла. Далее следует текст, состоящий из слов.
Для каждого найденного вхождения требуется вывести номер строки текста и номер слова в строке,
с которого начинается вхождение. Нумерация строк и слов начинается с единицы. Номер строки в тексте
отсчитывается от начала текстовой части, без учёта строки с образцом.
}
